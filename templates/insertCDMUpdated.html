The CIS Critical Security Controls (CSC), created by top experts from many organizations recommends set of actions to provide an efficient defense against pervasive and advanced attacks. CIS has combined attack patterns derived from threat reports with experts' defense knowledge to form a synthesized list of security controls.

Security function is the exact action executed by the security controls against threats.

The entities on which the security controls operate.

CCAA has generated CCAA Cyber Kill Chain by combining Cyber Kill Chain of Lockheed Martin and The Mitre Corporation which divides the adversary's Tactic, Technique and Procedure into eight different steps or phases for better understanding and analyzing. These phases are described as follows:

Reconnaissance: The intruder performs scanning, crawlong etc. to identify the target and also to discover the vulnerabilities of that victim target.

Weaponization: The intruder develops or prepares the weapon such as remote access malware to exploit the vulnerabilities of the target.

Delivery: The attacker delivers the weapon to the target via email, removable disk etc.

Exploit: The attacker exploits the identified vulnerabilities by executing the malicious code on target system.

Installation: The attacker installs malware,access point etc. on the victim machine.

Command & Control(C2): The attcker establishes a communication channel with the remote master server to enable the instruder to manipulate the target machine.

Execiute: The attacker executes action to achieve the goals such as data exfiltration, attack propagation etc.

Maintain: This can come at any phase after the exploitation. In this phase, the attacker performs actions to maintain its presence.
